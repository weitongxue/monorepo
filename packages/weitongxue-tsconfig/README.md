# @weitongxue/tsconfig

## 简介

`@weitongxue/tsconfig`为`ts`基础相关配置,具体配置参考源码

## 安装

```ts
npm install typescript @weitongxue/tsconfig --save-dev
```

## 配置文件`tsconfig.josn`

```json
// 客户端基础配置
{
  "extends": "@weitongxue/tsconfig/client",
};
```

```json
// web端基础配置
{
  "extends": "@weitongxue/tsconfig/web",
};
```
